Perguntas Frequentes

Tudo que você precisa saber sobre o TBG

Como ativo o teste grátis? +
Após instalar a extensão, abra o TBG no Gladiatus, vá na aba Sistema e clique em "Ativar Trial". Você terá 7 dias grátis para testar todos os recursos. O trial é vinculado ao seu Profile Code, então só pode ser usado uma vez por conta.
Como compro uma licença? +
Acesse a página de Planos, escolha o plano desejado, informe seu Profile Code e email. Você será redirecionado para o Mercado Pago, onde pode pagar com cartão, PIX ou boleto. Após a confirmação do pagamento, a chave de licença é enviada automaticamente para o email informado.
Quais navegadores funcionam? +
O TBG funciona em todos os navegadores baseados em Chromium: Google Chrome, Microsoft Edge, Brave, Opera, Vivaldi e Arc. Não funciona no Firefox.
É seguro usar? +
O TBG não coleta senha, email ou quaisquer dados de login. A extensão lê apenas o conteúdo da página do Gladiatus para executar as automações. O Profile Code é feito pelo próprio TBG para poder vincular a licença à sua conta após o pagamento do plano desejado. Os pagamentos são processados pelo Mercado Pago, portanto não temos nenhum acesso a seus dados, tudo é feito diretamente com o Mercado Pago.
Pode resultar em banimento? +
O uso de ferramentas de automação pode violar os Termos de Serviço do Gladiatus. O usuário assume total responsabilidade por quaisquer consequências impostas pela Gameforge, incluindo bloqueio ou suspensão da conta. O TBG é fornecido "como está", sem garantias.
Como funciona a licença? +
A licença é vinculada ao seu Profile Code do Gladiatus. Cada conta precisa de uma licença própria. A licença é validada automaticamente pelo servidor do TBG. Se você trocar de computador, basta instalar a extensão e sincronizar sua chave na aba Sistema.
Cada conta precisa de uma licença? +
Sim, cada conta do Gladiatus precisa de uma licença própria. Mas não se preocupe — oferecemos planos multi-conta com descontos exclusivos para 2 ou 3 contas. Acesse a página de Planos e clique em "Monte Seu Plano" para ver os preços. Para 4 ou mais contas, entre em contato via email.
Como obtenho meu Profile Code? +
Abra o TBG no Gladiatus, vá na aba Sistema. Seu Profile Code será exibido na tela. Copie-o e use na hora de comprar ou ativar sua licença.
Funciona em qualquer servidor do Gladiatus? +
Sim! O TBG funciona em todos os servidores do Gladiatus. A interface do TBG está disponível em três idiomas: Português, Inglês e Espanhol — você pode escolher o idioma dos menus nas configurações da extensão.
Tem reembolso? +
Como se trata de um produto digital com entrega imediata da chave de licença, não oferecemos reembolso após a entrega. Por isso, oferecemos 7 dias de teste grátis para que você avalie todos os recursos antes de comprar. Em casos de falha técnica comprovada, entre em contato para avaliação.
